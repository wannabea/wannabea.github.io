Poj 1201 [差分约束] + [Spfa]

发表于 2019-10-15 分类于 解题报告 评论数： 阅读次数：

题意

给出n行数据，每行三个数a,b,c,代表[a,b]的闭区间内，至少有c个点被标记，求被标记的最少的点数。

题解

1）转化

​ 我们设置一个s数组，s[i]可以代表第0 - i中被标记的点，则一组数据中a,b,c则表示为s[b] - s[a-1] >= c这样的话， s数组的下界就变成了s[-1],为了避免-1的出现，我们将所有的i+1,即s[i+1]代表0 - i中被标记的点数量。这时我们可以得到s[b+1] - s[a] >= c 我们将s[i]看作一个点，c的值为 b+1 - a的权值，我们要求的，也就是最大的b到最小的a点之间，存在多少被标记的点.

2）建图

​ 我们首先要了解差分约束的概念，加入我们有三个式子，

​ A - B >= 2

​ B - C >= 3

​ A - C >= 4

​ 就可以由此建图如下

​ 可见A-C的最小值为5，这时，我们可以联想到什么呢，最短路，图的建立与图的结果与最短路很相似，我们就可以使用最短路的方法来求最长路，就只需要将判断的符号改变。

​ 针对本题，仅有的n个条件是不足以将所有的点连接成连通图的，但是，还有个隐含条件就是s[i]-s[i-1] <= 1 s[i]-s[i-1] >= 0 这时我们就可以将所有点连接成连通图。

3）求最长路

​ 在差分约束系统中，常常存在负环，这时，使用dijkstra是无法得到结果的，我们就需要使用bellman_ford或者spfa，对于spfa(毕竟spfa更简单),cnt数组记录入队次数，若超过所有点的个数，则表明无法得到结果及时跳出循环。最终的结果就是，spfa从最小点mi出发，在mx结束中存留的结果。
